Abstract

This project covers a detailed analysis of the Titanic dataset to uncover patterns and insights related to passenger survival. Using data preprocessing, exploratory data analysis (EDA), and a machine learning model, it demonstrates significant factors influencing survival. A Random Forest Classifier was used to predict outcomes with notable accuracy.

Introduction

The Titanic dataset provides valuable insights into the factors that influenced passenger survival. This analysis included data preprocessing, EDA, feature engineering, and building a predictive model. This project aims to refine data analysis skills and gain practical machine learning experience.
